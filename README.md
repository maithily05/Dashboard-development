# Dashboard Development - Internship Project

**Company:** SolverMinds Technologies Private Limited 
**Duration:** 06/2025 - 07/2025  
**Tool Used:** SEDGE - DashPro

## Overview
Created interactive dashboards to visualize key metrics and automate reporting.  
The dashboards were designed to monitor marine operations and logistics metrics, track performance, and support decision-making for specific departments.
Focus was on usability, data clarity, and real-time insights.

## Features
- Real-time data visualization  
- Automated reporting and alerts  
- Interactive filters and drill-downs  
- Clean, user-friendly interface
